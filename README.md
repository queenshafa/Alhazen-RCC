# Alhazen RCC – Ramadhan Portal

**Ramadhan Portal** is a website that allows users to track their daily Ramadhan days easily. Featuring imsak and adzan schedule, daily prayers recitation, zakat kalkulator, daily tracker (Qur'an reading, shalat tracker, etc), and a digital tasbih.

## 🚀 Features
* **Imsakiyah Schedule:** A feature where users can check the adzan time of their city.
* **Prayers Recitation:** Users can read the prayer recitation of iftar.
* **Zakat Calculator:** A calculator to calculate the charity users will give for Eid.
* **Daily Tracker:** A tracker used to track users' daily activities (Qur'an reading, Shalat, and Fasting tracker).
* **Digital Tasbih:** A digital counter, used to count users' tasbih. Also, come along with targets.

## 🛠️ Tech Stack
* **Structure:** HTML5
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Logic:** JavaScript
